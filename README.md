# 🐾 Feline Society - AstroShop Admin Panel

**Feline Society** es una plataforma de gestión administrativa diseñada para **AstroShop**, una tienda especializada en equipos de astronomía con una identidad visual inspirada en la calidez y ternura de los gatos. El sistema permite la gestión de usuarios, inventarios y reportes mediante una interfaz intuitiva y acogedora.

## 🚀 Tecnologías Utilizadas
* **HTML5:** Estructura semántica de la plataforma.
* **CSS3:** Diseño personalizado con enfoque en "Soft UI", bordes redondeados y paleta de colores pastel.
* **Vibecoding:** Metodología de desarrollo asistido por IA para prototipado rápido.
* **Google Fonts:** Uso de 'Chelsea Market' para un toque amigable y 'Dancing Script' para detalles elegantes.

## 📂 Estructura del Proyecto

```text
/
├── index.html                # Página de aterrizaje principal
├── IMG/                      # Recursos visuales (Logos, Avatares, Iconos)
└── ADMIN/
    ├── index-admin.html      # Panel principal del administrador (Dashboard)
    ├── CSS/                  # Hojas de estilo
    │   ├── estilos-admin.css # Estilos globales del panel
    │   └── estilos-crear.css # Estilos específicos para formularios
    ├── HTML/                 # Módulos funcionales
    │   ├── MOD_1_USUARIOS/   # Gestión de usuarios (Crear, Consultar, Reportes)
    │   ├── MOD_2_INVENTARIO/ # Gestión de productos astronómicos
    │   └── MOD_3_SISTEMA/    # Configuración y perfiles
    └── JS/                   # Lógica de interacción (Próximamente)
```
